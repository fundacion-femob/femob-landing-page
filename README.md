# FEMOB - LANDING PAGE

## Development Setup

Clone de repo:

``` bash
$ git clone ssh://git@git.rabobank.nl:7999/sk/mui-crzb-is-spa.git
```
Install the Angular CLI
```
$ npm install -g @angular/cli
```

## Serve the application:
```
$ ng serve --open
```

## Page URL:
```
https://femob-landing-page.firebaseapp.com
```
